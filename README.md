# Global Car Sales Performance - Power BI Dashboard

## Project Overview
This project is an interactive, multi-page Power BI dashboard designed to analyze global car sales data. The objective is to provide a comprehensive view of the automotive market, tracking the performance of major car manufacturers worldwide and delivering actionable insights through data visualization.

## Tools & Technologies Used
* **Power BI Desktop:** For data visualization, building interactive charts, and dashboard design.
* **Power Query:** For data cleaning and transformation.
* **DAX (Data Analysis Expressions):** For creating custom calculated columns and measures (KPIs).

## Dashboard Structure
The dashboard is structured into 11 distinct pages, providing both high-level market overviews and granular, brand-specific analysis. Navigation buttons are implemented for a seamless user experience.

* **Welcome Page:** Project introduction and navigation menu.
* **Analysis Performance:** Overall global market KPIs, total sales, and market share comparisons.
* **Brand-Specific Pages (9 pages):** Detailed individual performance analysis for:
  * Audi
  * BMW
  * Ford
  * Toyota
  * Nissan
  * KIA
  * Hyundai
  * Mazda
  * Mitsubishi

## Key Insights
* Comparative analysis of sales volume across top global brands.
* Identification of top-performing car manufacturers in different segments.
* Trend analysis of market share and brand dominance.

## How to Run This Project
1. Clone this repository or download the files.
2. Ensure you have **Power BI Desktop** installed on your machine.
3. Open the `Global_Car_Sales.pbix` file to interact with the dashboard and view the underlying data model.
